#Tic-Tac-Toe with Minimax AI

A command-line Tic-Tac-Toe game built in C++ where players compete against an unbeatable AI powered by the Minimax algorithm.

#Features\
🎮 Human vs AI gameplay\
🧠 Minimax algorithm for optimal AI decisions\
✅ Win, draw, and move validation\
📟 Simple console interface\
⚡ Recursive game-tree search\
#Concepts Used\
Recursion\
Minimax Algorithm\
Backtracking\
Game Tree Search\
Vectors\
Functions\
Decision Making (AI)\
Technologies\
C++\
Standard Template Library (STL)\

This project was built to learn the fundamentals of game AI and recursive search algorithms while implementing a classic strategy game.
